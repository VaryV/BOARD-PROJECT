CS BOARD PROJECT DOCUMENTATION FORMAT

FONT SIZE = 12(TEXT), 14/16(HEADINGS, SUBTITLES, ETC.)
FONT = Times New Roman

1. COVER PAGE
    i.   TITLE/PROJECT NAME (COMPUTER SCIENCE PROJECT FOR 2020-2021)
    ii.  APPROPRIATE IMAGE FOR PROJECT
    iii. NAME
2.  INDEX
3.  ACKNOWLEDGMENT
4.  BONAFIDE
5.  CASE STUDY OF THE PROJECT(DOMAIN,MODULES USED, DESCRIPTION OF THE PROJET)
6.  SCREENSHOTS OF TABLES OR DESC OF MySQL TABLES USED
6.  DRAW FLOWCHART(DIAGRAMATICALLY EXPLAIN HOW THE PROJECT WORKS)
7.  SOURCE CODE(THE PYTHON CODE THAT IS USED)
8.  SAMPLE SCREENSHOTS(FEW(3-4)(2-3 PAGES) IMPORTANT IMAGES OF HOW THE PROJECT WORKS)
9.  LIMITATIONS OF THE PROJECT
10. SCOPE FOR IMPROVEMENT
11. BIBLIOGRAPHY
